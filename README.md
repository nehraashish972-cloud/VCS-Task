# VCS Task - Git Commands Practice


## Overview
This task demonstrates how to create a local Git repository, 
push it to GitHub, and perform Merge, Rebase, and Stash operations.

## Tools Used
- WSL (Ubuntu 22.04)
- Git
- GitHub

## Steps Performed

### 1. Directory & Scripts Setup
- Created VCS-Task directory
- Created script1.sh, script2.sh, script3.sh

### 2. Git Init & Push
- Initialized local Git repository
- Linked to GitHub remote
- Pushed code to main branch

### 3. Merge
- Created feature-branch
- Added feature.sh
- Merged feature-branch into main

### 4. Rebase
- Created rebase-branch
- Added rebase-script.sh
- Rebased onto main branch

### 5. Stash
- Created stash-test.sh
- Used git stash to save temporarily
- Recovered with git stash pop
- Committed and pushed
